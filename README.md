# 容疑者AIの供述：矛盾を探せ

Claude AIが演じる容疑者たちを尋問し、供述の矛盾を見抜いて真犯人を特定するミステリー推理ゲーム。

## ゲーム概要

- **プレイ時間**: 5〜15分
- **プレイ人数**: 1人
- **対象**: IT知識不要、誰でもプレイ可能

### ゲームの流れ

1. **スタート画面** - プレイヤー名を入力
2. **イントロ** - 事件のあらましと容疑者3人のプロフィールを確認
3. **尋問パート** - 最大15ターンで容疑者にチャットで質問し、矛盾を探す
4. **結果発表** - 犯人を指名し、スコアとランキングを確認

## 技術スタック

- **フロントエンド**: React 19 / TypeScript / Tailwind CSS 4
- **AI**: Anthropic Claude API (claude-sonnet-4-20250514)
- **ビルドツール**: [Vite+](https://vite.dev/) (Vite, Rolldown, Vitest, Oxlint, Oxfmt)
- **パッケージマネージャ**: pnpm（Vite+ 経由で使用）

## セットアップ

### 1. 環境変数の設定

```bash
cp .env.example .env
```

`.env` に Anthropic API キーを設定:

```
VITE_ANTHROPIC_API_KEY=your-api-key-here
```

### 2. 依存関係のインストール

```bash
vp install
```

### 3. 開発サーバーの起動

```bash
vp dev
```

## プロジェクト構成

```
src/
├── api/          # Claude API クライアント
├── components/   # UIコンポーネント（チャット、キャラクター表示、スコアボード等）
├── data/         # シナリオデータ、不安キーワード辞書
├── hooks/        # カスタムフック（ゲーム状態、API通信、タイプライター演出）
├── screens/      # 画面コンポーネント（スタート、イントロ、尋問、結果）
├── state/        # React Context + Reducer によるゲーム状態管理
├── types/        # TypeScript 型定義
└── utils/        # スコア計算、プロンプト構築、ストレージ等のユーティリティ
```

## ゲームルール

### スコアリング

| 項目         | ポイント     |
| ------------ | ------------ |
| 基本スコア   | 1000         |
| ターン消費   | -50 / ターン |
| ヒント使用   | -150 / 回    |
| 正解ボーナス | +500         |

### ランキング

| ランク | スコア    | 称号       |
| ------ | --------- | ---------- |
| S      | 1200+     | 天才探偵   |
| A      | 900〜1199 | 優秀な刑事 |
| B      | 600〜899  | 一般捜査官 |
| C      | 300〜599  | 見習い探偵 |
| D      | 0〜299    | 迷探偵     |

### ヒントシステム

同じ容疑者に5回以上質問するとヒントボタンが解放されます。使用するとスコアが150ポイント減少します。
